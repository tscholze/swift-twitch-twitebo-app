# Twitebo - Twtich Teams Browser
> Uses the Twitch API to get information from a team and it's members. It also allows to watch an online team's streamer.

## Prerequirements
- Xcode 10.2+
- Swift 5
- swiftlint
- swiftformat
- Twitch Developer API Client Id
